# 绘制一个思维导图

根据用户提供的需求绘制思维导图。

绘制思维导图可以使用MCP服务：`simple-mind-map-mcp`。如果已经安装了这个服务，那么你可以根据下面的流程来进行。如果没有安装，那么这个技能无法使用。

## 工作流程

### 第一步

调用服务的`start_editor`工具，该操作会在浏览器打开一个新标签页，显示思维导图编辑器界面，并渲染默认思维导图数据。

### 第二步

如果用户提供了要绘制为思维导图的具体内容，你需要先判断是否是`Markdown`数据。如果是，那么直接调用服务的`set_mind_map_data`工具来绘制思维导图。如果不是，需要提示用户输入正确的`Markdown`数据。

如果用户没有提供具体的思维导图内容，但是提供了思维导图的主题，那么你需要根据主题生成`Markdown`数据，然后调用服务的`set_mind_map_data`工具来绘制思维导图。

如果用户既没有提供内容，也没有提供主题，那么你需要追问用户，询问他要绘制的思维导图的主题。

可选：

1.如果用户指定了要使用什么结构，那么需要根据用户的指定结构在可选结构中选择一个最符合用户需求的结构。否则，该参数不需要传递。

2.如果用户指定了要使用什么样式主题，那么需要根据用户的指定主题在可选主题中选择一个最符合用户需求的主题。否则，该参数不需要传递。

设置完思维导图数据后，接下来可以仅更新思维导图的样式主题或结构。

仅更新样式主题可以调用服务的`set_mind_map_theme`工具。

仅更新结构布局可以调用服务的`set_mind_map_layout`工具。

注意：

1.如果用户没有显示指定读取当前项目文件，否则不要进行该操作。

2.从始至终无需在本地生成任何文件。

## 所有可用工具

- `start_editor` - 启动思维导图编辑器
- `set_mind_map_data` - 全量更新编辑器渲染的思维导图数据
- `set_mind_map_theme` - 修改编辑器渲染的思维导图样式主题
- `set_mind_map_layout` - 修改编辑器渲染的思维导图结构布局
- `close_editor` - 关闭当前打开的思维导图编辑页面
- `export_mind_map_to` - 将当前渲染的思维导图导出为指定类型的文件
